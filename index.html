Colorful Shapes Screensaver

Fall 2013

I got bored one night and thought I'd make a screensaver. Spawns shapes of varying colors, each of which has a different radial growth speed. When two shapes collide, their growth speed changes to shrinking speed, until they hit a radius of 0. Shapes are spawned relatively near each other using a probabalistic algorithm that tends to make shapes spawn in emptier regions.

  • Github

Techfair Kinect App

Spring 2013

This app was made for advertising MIT Techfair. Initially, particles will spell out "techfair" (but this can change to any string, with any font). Raising one hand will make all particles follow that hand; raising two hands will split the particles into two. Dropping both hands will cause the particles to rearrange to the spelling.

Minesweeper Proof of Concept Hack

Splash 2012

This project is a proof-of-concept for my Splash class on how to hack Minesweeper with assembly. After the first mine is clicked (because Minesweeper doesn't generate the board until then), the program can be run and will print the board (with mines). Note: only tested on 64 bit Windows 7.

Louisiana Mu Alpha Theta Website

Summer 2012

After serving as the webmaster, the club sponsors and I decided that the old website needed to be redone. During that convention, I wrote a lot of queries by hand on the fly to get information (e.g., how many students are taking X test). They contracted me to rewrite the website so that convention is much more easily run. Sponsors can view reports about different tests, generate name tags, and do other convention duties. School sponsors can also register their students for convention.
